O que é Frontend?
O Frontend é a parte de uma aplicação web (ou sistema) com a qual o usuário interage diretamente. É a interface gráfica, composta por tudo o que você vê e clica em uma tela, como botões, menus, textos, imagens e formulários.

Cliente x Servidor
Cliente: É o dispositivo do usuário (computador, celular) e o navegador (Chrome, Firefox) que faz pedidos para acessar um site. Ele consome as informações e renderiza o Frontend.
Servidor: É um computador potente e remoto que armazena os arquivos do site, processa os dados (Backend) e envia as respostas de volta para o cliente.

Exemplo prático: Quando você digita google.com no seu navegador (Cliente), você está fazendo um pedido. O Servidor do Google recebe esse pedido, processa e envia os arquivos visuais da página de volta para o seu navegador exibir.

O papel de HTML, CSS e JavaScript
HTML (HyperText Markup Language): É a estrutura da página. Funciona como o "esqueleto" do site, definindo onde ficam os textos, títulos, imagens e links usando tags.
CSS (Cascading Style Sheets): É o visual e o estilo. Funciona como a "roupa e maquiagem" do site, definindo cores, fontes, espaçamentos, alinhamentos e layout.
JavaScript: É o comportamento e a interatividade. Funciona como os "músculos e o cérebro", permitindo que a página reaja a ações do usuário (como abrir um menu lateral, validar um formulário ou carregar dados sem atualizar a tela).

O que é W3C?
O W3C (World Wide Web Consortium) é um consórcio internacional fundado por Tim Berners-Lee (o criador da Web) que tem como objetivo principal desenvolver padrões e diretrizes para a internet.
Eles garantem que tecnologias como HTML e CSS sigam regras universais para que qualquer site funcione da mesma forma em qualquer navegador e em qualquer dispositivo no mundo, promovendo também a acessibilidade na web.
