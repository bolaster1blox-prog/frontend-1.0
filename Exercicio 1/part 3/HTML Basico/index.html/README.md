# Explicação das Tags HTML Utilizadas

Este documento explica brevemente a função de cada tag utilizada no exercício de HTML básico.

* `<!DOCTYPE html>`: Informa ao navegador que o documento atual é um arquivo HTML5.
* `<html>`: A tag raiz que envolve todo o conteúdo da página web.
* `<head>`: Contém os metadados da página (como configurações de caracteres e o título), que não aparecem diretamente no corpo do site.
* `<meta charset="UTF-8">`: Define a codificação de caracteres como UTF-8, permitindo exibir corretamente acentos e caracteres especiais.
* `<meta name="viewport">`: Configura a visualização da página para que ela se ajuste corretamente a diferentes tamanhos de tela (responsividade).
* `<title>`: Define o título da página que aparece na aba do navegador.
* `<body>`: Contém todo o conteúdo visível da página, como textos, imagens e listas.
* `<h1>`: Define o título principal da página. Deve ser usado apenas uma vez por página para manter uma boa hierarquia.
* `<h2>` e `<h3>`: São subtítulos utilizados para organizar e seccionar o conteúdo em níveis de importância.
* `<p>`: Define um parágrafo de texto comum.
* `<ul>`: Cria uma lista não ordenada (geralmente exibida com marcadores de ponto/bullet points).
* `<ol>`: Cria uma lista ordenada (exibida numericamente em sequência).
* `<li>`: Define um item individual dentro de uma lista (tanto para `<ul>` quanto para `<ol>`).
